Term

Definition
Machine Language: Encoding of instructions into binary so they can eb executed by the computer.
Debugging: Using a program to find the issue in a computer program.
Assembler: A program that converts patterns of bits so it can perform basic operations.
Assembly Language: A programming language that communicates directly with a computers harddware.
Translator: Program that converts source code into object code.
Compiler: Trnaslates a source code into another type of programming language.
Interpreter: Translates source code into object code one instruction at a time.
Cross-Platform Software: A type of application that works on various devices called platforms.
Natural Language: The ability of computers to understand text and words.
Formal Language: A set of strings of symbols from a finite alphabet.
Machine Independence: Software that is not dependent on the properties of a machine.
High Level Programming Language: Any programming language that enables developemnt of a program.
Object Oriented Programming (OOP): A style of programming by identification of classes of objects linked with methods.
Scripting Language A series of commands a software interprets one at a time.
