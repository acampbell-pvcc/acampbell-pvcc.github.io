All About GitHub

  1. What is the version control system?
    The practice of tracking and managing changes to software code.
  2. What is Git?
    A free and open source version control system designed to handle everything from small to large projects.
  3. What is GitHub?
    An internet hosting service for software development and version control using Git.
  4. What is the difference between Git and Github?
    Git allows you to manage and keep track of your code history. Github is a cloud based hosting service that allows you to manage your Git repositories.
  5. Who started Github and how was it started?
    Github was developed by Chris Wanstrath, P.J. Hyett, Tom Preston-Werner and Scott Chacon using Ruby on Rails due to their need.
  6. What company owns it now?
    The Microsoft Corporation
  7. How much does a GitHub account cost now?
    GitHub offers free accounts or a monthly fee of $40 to $120.
  8. What is the Octocat?
    The mascot of GitHub.

Git/Github Terms

  1. Repository: Computer storage for maintaining data or software packages.
  2. Commit:The making of a set of tentative changes permanent , marking the end of a transaction and providing durability to ACID transactions.
  3. Fork:A system used to create processes.
  4. Push: A way to set your data segment to the same value as your code segment.
  5. Pull Requests:An event that takes place in software development when a developer is ready to begin the process of merging new code changes with the main project repository.
  6. Workflows:Sequences of tasks that process a set of data through a specific path from initiation to completion.
  7. Issues: A few issues with hardwares are hacking, privacy, data collection, algorithmic bias, and more.
  8. Raw Button:The keyboard driver reports these codes directly to applications when the keyboard is in raw mode.
  9. Blame Button:Used to examine the contents of a file line by line and see when each line was last modified and who the author of the modification was.
