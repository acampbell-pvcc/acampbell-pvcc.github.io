Term

Definition
Network Classifications:
  • LAN: A local area network that consists of a series of computers linked together to form a network in a specific location.
  • MAN:A metropolitian area network is a computer network that connects computers within metropolitan areas.
  • WAN:A wide-area network is technology that connects offices, data centers, cloud applications and could storage together.
Network Models:
  • Bus Network: A LAN topology in which node, a device, is connected to a main cable or link called a bus.
  • Star Network: A LAN topology in which personal computers, workstations or other devices are directly connected to a common central computer that is reffered to as a hub.
Protocols:
  • CSMA/CD Ethernet:Carrier-sense multiple access with collision detenction is a medium access control method used mostly in early ethernet technology for local area networking.
  • CSMA/CA Wifi:A protocol for carrier transmission in networks. Prevents potential of a collision occuring when two of more stations send their signals over data.
Combining Networks:
  • Repeater:Expands the coverage area of the network, repropgate a weak or broken signal and or service remote nodes.
  • Bridge:A type of computer network device that provides interconnection with other bridge networks that use the same protocol.
  • Switch:A device that channels incoming data from any of multiple input ports to the soecific output port that will take the data toward its intended destination.
  • Router:A networking device that connects computing devices and networks to other networks.
  • Gateway:A computer that sits between different networks or applications.
Client/Server Model:
  • P2P:Peer-to-peer networking, a group of computers are linked together with equal permissions and responsibilities for processing data.
  • Distributed System:A collections of independent components located on different machines that share messages with each other in order to achieve common goals.
  • Cloud Computing:Networks, data storage, applications, security and development tools are enabled via the internet.
Internet Addressing:
  • Tier 1:An internet protocol network that can reach every other network on the internet solely via settelment-free interconnection.
  • Tier 2: An internet service provides which engages in the practice of peering with other networks, but which also purches IP transit to reach another portion of the internet.
  • Tier 3:Primary engagement of delivering internet access to end customers.

Internet Protocols:
  • FTP:File transport protocol is a network protocol for transmitting files between computers over TCP/IP connections.
  • HTTP: Hypertext transfer protocol is an application protocol for distributed, collaborative, hypermedia information ststems that allows user to communicated data on the web.
  • SMTP:Simple mail transfer protocol is used to send and recieve mail.
World Wide Web:
  • URL:Used to locate a resource on the internet.
  • Client-Side Activities: Everything in a web application that is displayed or takes place on the clients device.
  • Server-Side Activities:Uses databases to store and retieve data and sending the correct data to the client as requested.
Internet Four-Layer Software:
  • Application Layer: An abstraction layer that specifies the shared communication protocols and interface methods used by hosts in a communications network.
  • Transport Layer:Provides communications between application processes running on different hosts within a layered architecture of protocols and other network components.
  • Network Layer: Part of the internet communications process where these connections occur by sending packets of data back and fourth between different networks.
  • Link Layer:Protocol layer in a program that handles the moving of data into and out of a physical link in a network.
Internet Is:
  • Packet-Switching Network:The transfer of small pieces of data across various networks.
  • Fault Tolerant: The ability of a system to continue operating without interruption when one or more of its components fail.
  • Redundant:A system design in which a component is duplicated so if it fails there will be a backup.
Cybersecurity:
  • Malware: A file or code that infects or steals virtually anything an attacker wants.
  • Virus: A program that can infect a computer without permission or knowledge of the user.
  • Worm:A subset of the trogan horse malware that can propogate or self-replicate from one computer to another without human activitation.
  • Trojan Horse:Malware that disguises itself as a legitimate code or software.
  • Spyware:Any software that installs itself on a computer and starts covertly monitoring your online behavior without your knowledge or permission.
  • Phishing:A scam where thieves steal personal or financial account information by sending deceptive electronic messages.
  • DDoS Attack:Involves multiple online connect devices, which ar eused to overwhelem a target website with fake traffic.
Protection:
  • Firewall: A network security device that monitors and filters incoming and outgoing network traffic based on an organizations previously established security policies.
  • Spam Filter: A program used to detect unsolicited, unwated and virus infected emails and prevent messages from coming into the inbox.
  • Proxy Server:Acts as an intermediary between an endpoint device, like a computer and another server from which a user or client is requesting a service.
  • Antivirus Software:Created specifically to help detect, prevent and remove malware.
  • Cryptography:Secure information and communication techniques derived from mathematical concepts and a set of rule based calculations called algorithims to transform messages in ways that are harder to decipher.
  • Public Key Encryption:Method of encrypting or signing data with different keys and making one of the keys or public key available for anyone to use.
