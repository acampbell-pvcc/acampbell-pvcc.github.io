Chapter 3 Terminology Table

Term
 Definition
Operating System: The software that controls the overall operation of the computer including file management, the user interface, and the program execution environment. Job Queue: A data structure maintained by job scheduler software containing jobs to run. FIFO: An acyonym for first in, first out. A method for organizing and manipulating data buffers as a queue. Interactive Processing: In a system, processing where a computer program or a procedure directs the system to accept and respond to input. In a computer system, processing allows each used to cause a response in the computer program. Time-Sharing: Allows a central computer to be sahred by a large number of users sitting at terminals. Multiprogramming: A form of parallel processing in which several programs run at the same time on a uniprocessor system. Application Software: A type of computer program that performs a specific personal, educational, and business function. System Software: A program designed to run a computer's hardware and applications and manage its resources, such as its memory, processors and devices. Utility Software: A system that includes all systems and programs on a computer system that maintains its functionality. Graphical User Interface(GUI): A graphics based operating system interface that uses icons, menus, and a mouse )to click on the icon or pull down the menus) to manage interaction with the system. Shell: Presents a command line interface which allows you to control your computer using commans entered with a keyboard instead of controlling graphical user interfaces (GUI) with a mouse, keyboard and touchscreen combination. Kernel: The essential foundation of a computers operating system. Device Driver: A driver or device driver, a set of files that tells a piece of hardware how to function by communicating with a computers operating system. File Manager: A computer program that provides a user interface to manage files and folders. Memory Manager: An operating system that handles the main memory of a computer. Virtual Memory: A common technique used in a computers operating system. Uses both hardware and software to enable a computer. Boot Strapping: To boot or load a program into a computer using a much smaller initial program to load in the desired program. ROM: Read only memory, a type of computer storage containing no-volatile, permanent data that normally can only be read, not written. Deadlock: A situation in which two computer programs sharing the same resource are effectively preventing each other from accessing the resource, resulting in both programs ceasing to function. Spooling: A process in which data is temporarily held to be used and executed by a device, program or the system. Super User (Or Administrator): A special user account used for system administration. Multi-Factor Authentication: An authentication method that requires the user to provide two or more verification factors to gain access to a resource such as an application, online account, or VPN.
